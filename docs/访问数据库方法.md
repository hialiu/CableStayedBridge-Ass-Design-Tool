## 如何给服务器传输文件

这里采用cloudreve的方式

- 在浏览器输入并访问 47.102.209.81:5212
- 用邮箱注册一个账号
- 向cloudreve上传文件
- 在服务器打开以下文件夹即可找到自己上传的文件

```
C:\Users\Administrator\Desktop\cloudreve_3.3.2_windows_amd64\uploads
```


- 服务器给本地电脑传文件将文件拖入对应文件夹即可
